# Лабораторная работа: Регулярные выражения в C#

## Описание

Работа демонстрирует использование регулярных выражений в C# с помощью класса `Regex` из пространства имён `System.Text.RegularExpressions`. В качестве тестового текста используется стихотворение М. Ю. Лермонтова «Парус».

---

## Задачи

- [x] Найти отрывок текста
- [x] Подсчитать количество букв, слов и конкретных словосочетаний
- [x] Вывести строки, начинающиеся с определённого символа
- [x] Вывести строки, оканчивающиеся определённым символом
- [x] Изменить часть текста

---

## Используемые паттерны

| Паттерн | Описание | Пример |
|---|---|---|
| `[а-яёА-ЯЁ]` | Любая русская буква | Подсчёт букв |
| `\b[а-яёА-ЯЁ]+\b` | Целое русское слово | Подсчёт слов |
| `\bон\b` | Точное слово «он» | Подсчёт вхождений |
| `^И.+$` | Строка, начинающаяся с «И» | Фильтрация строк |
| `^.+!$` | Строка, оканчивающаяся на «!» | Фильтрация строк |

---

## Структура кода

```
Program.cs
├── Исходный текст (стихотворение «Парус»)
├── Подсчёт символов и слов
│   ├── Количество букв
│   ├── Количество слов
│   └── Вхождения слова «он»
├── Строки, начинающиеся с «И»
├── Строки, оканчивающиеся на «!»
└── Замена «он» → «ОН»
```

## Использованные классы и методы

- `Regex.Matches(text, pattern)` — поиск всех совпадений, возвращает `MatchCollection`
- `Regex.Replace(text, pattern, replacement)` — замена по шаблону
- `RegexOptions.Multiline` — `^` и `$` соответствуют началу/концу каждой строки
- `RegexOptions.IgnoreCase` — игнорирование регистра
