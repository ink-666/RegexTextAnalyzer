# RegexTextAnalyzer

## Описание
Программа на C# для анализа текста с использованием регулярных выражений.

## Функциональность
- Подсчёт символов и слов в тексте
- Поиск строк, начинающихся с определённого слова
- Поиск строк, оканчивающихся определённым символом
- Замена частей текста по шаблону

## Использованные технологии
- C# .NET
- Регулярные выражения (System.Text.RegularExpressions)
