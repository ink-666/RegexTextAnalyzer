# Регулярные выражения в C#

## Описание

Программа демонстрирует работу с регулярными выражениями в C#:
- поиск слов по шаблону
- подсчёт символов и слов
- поиск строк по началу/концу
- замена текста

## Запуск
```
dotnet run
```
Пример вывода

```
Слова с корнем 'губ':
  тупогуб
  тупогубенький
  губа

Количество букв и цифр: 87
Количество слов: 18

Строки, начинающиеся с 'Бык':
  Бык тупогуб, тупогубенький бычок, у быка губа бела была тупа

Только цифры из телефона: 71234567890
Используемые регулярные выражения
```

```
Выражение	Назначение
\w*губ\w*	Поиск слов с корнем "губ"
[A-Za-zА-Яа-я0-9]	Поиск букв и цифр
\b\w+\b	Поиск слов
^\w+	Строки, начинающиеся с буквы
\.$	Строки, заканчивающиеся точкой
\s+	Один и более пробелов
\D	Любой нецифровой символ
```
